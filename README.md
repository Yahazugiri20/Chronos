# 🏛️ Chronos: AI-Powered Historical Integrity

**Chronos** is an autonomous AI Agent designed to verify and archive historical claims on the blockchain. By combining LLM-based research with the immutability of the **Base Network**, Chronos ensures that historical truth remains tamper-proof in the age of generative AI.

## 🚀 Features
- **Autonomous Verification:** AI-driven analysis of historical data.
- **On-Chain Attestation:** Permanent record storage on Base Sepolia.
- **Historical Integrity:** Specialized logic focused on source criticism (Islamic Civilization History methodology).

## 🛠️ Technical Stack
- **AI Logic:** Python-based Agent.
- **Blockchain:** Solidity Smart Contract deployed on **Base Sepolia**.
- **Network:** [Base](https://base.org)

## 📦 Installation
1. Clone this repo.
2. Setup `.env` with your `PRIVATE_KEY` and `RPC_URL`.
3. Run `pip install -r requirements.txt`.
4. Execute `python agent.py`.
